This is a concept project to build a drag and drop inventory system in unity using with unity ui components.

![inventory](./images/config.png)

Implemented so far:
- Drag and drop slots
- Easy configuration with a scriptableobject file
- Tooltip
- Swap slots
- Stacable items
- Vendor

Todo features:
